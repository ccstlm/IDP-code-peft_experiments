# PEFT Experiments

This repository is a clean workspace for new PEFT and traffic representation experiments.

## Layout

- `src/peft_experiments/`: reusable experiment code
- `scripts/`: runnable command-line utilities
- `configs/`: experiment configuration files
- `experiments/`: experiment-specific entry points and notes
- `analysis/`: result analysis scripts and notebooks
- `docs/`: experiment reports and documentation
- `data/`: local data placeholders; raw datasets are not tracked
- `outputs/`: local run outputs; generated artifacts are not tracked

## Git Policy

Track source code, configs, reports, and small metadata files. Keep raw datasets, checkpoints, logs, and generated outputs local unless explicitly needed.
